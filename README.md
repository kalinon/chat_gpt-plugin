# ChatGPT Crystal Plugin

Crystal example of a Plugin for ChatGPT. Logic is based on the [plugins-quickstart](https://github.com/openai/plugins-quickstart) example.

## Installation

```sh
shards install
shards run
```

## Usage

<https://platform.openai.com/docs/plugins/>
